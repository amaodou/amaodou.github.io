vscode 安装 Settings Sync 插件同步设置

发表于 2020-07-07 更新于 2020-07-24

github 生成 Access Token

  1. 登录 github 进入Settings / Developer settings配置界面，点击Generate new token

  2. 生成新的 Access Token，勾选 gist

  3. 记录生成的 Access Token

vscode Settings Sync 插件配置

  1. 登录 github 并列出所有 gists，选择一个 gists 同步数据（如果还没有 gists 的话会自动创建一个）

  2. 打开 vscode Settings Sync 配置页，输入 Gist ID（执行第 1 步后会自动填上），输入令牌（即 github Developer settings生成的 token）

登录 github 查看同步结果
